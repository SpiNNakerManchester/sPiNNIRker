These pages document the python code for the spiNNIRker_ module
which is part of the SpiNNaker_ Project.

The spinnirker_models_ module contains the documentation of the C code for this module.

This code depends on SpiNNUtils_, SpiNNMachine_, SpiNNMan_, PACMAN_, SpiNNFrontEndCommon_ (Combined_documentation_).
